# evnio

A lightweight, fast non-blocking TCP network library based on Reactor mode.

# Feature

- fast and simple
- low memory usage
- cross-platform

# Getting Started

## Install

  go get -u github.com/dreamans/evnio
  
## Websocket example

  https://github.com/dreamans/evnio/tree/master/examples/websocket
